# Staging.bloxd.io (Scratch Edition)

This repository contains a **Scratch game exported with TurboWarp**, made as a *staging / experimental edition of someone else's recreation of bloxd.io*.

Because of file size limits, the project is distributed using **GitHub Releases** instead of GitHub Pages.

---

## Important Notice
The game itself is **over 50MB**. If you believe your device cannot handle the game, do not download it or play it.

---

## Downloads

You can download the game from the **Releases** section:

- **Single-file HTML build**  
  - Packaged into **one HTML file** with **Turbowarp**
  - Easy to run
  - Best option for most users

- **ZIP source build**  
  - Contains all assets and project files
  - Useful for inspection, modding, or re-hosting

 Go to the **Releases** tab to download the latest version.
 link: <https://github.com/Drag00nKnight999/Staging.bloxd.io-Scratch-Edition/releases>

---

## How to Play

### Option 1: Single HTML file (recommended)
1. Download the `.html` file from the release
2. Open it in your web browser
3. Click the screen to enable sound (audio requires interaction)

### Option 2: ZIP version
1. Download and extract the ZIP file
2. Open `index.html` in a browser
3. Make sure all files stay in the same folder structure
